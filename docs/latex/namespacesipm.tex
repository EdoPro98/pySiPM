@biref that calls all the procedures defined in libs to generate a complete Si\+PM event. 

This function is the main function used to simulate a complete Si\+PM event. It calls all the other methods with the correct parameters and user settings.


\begin{DoxyParams}{Parameters}
{\em times} & This list contains the arriving time of each photon on the Si\+PM sensor surface. This list is the main input of the simulation.\\
\hline
{\em other} & This variable may contain other informations about the event generated. It can be the event id, the arriving time inserted in the simulation or the real number of photons inserted in the simulation. This tuple will be copied as it is in the output.\\
\hline
\end{DoxyParams}

other The copy of the {\ttfamily other} variable given in the input.
